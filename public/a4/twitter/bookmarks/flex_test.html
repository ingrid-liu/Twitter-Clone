Case 3: Margin or Padding?

All items have the same layout by default: margin -> border -> padding -> content of this item

Basically, it means that if you want spacing inside of a visible border, use Padding, otherwise, use Margin.

Sometimes the padding spacing will not be visible if there's no space left for padding (e.g., when you speficy width / height in pixels while there're too much content in the container). In that case, use Margin instead as Margin will never be "covered" or "squeezed" by content.
